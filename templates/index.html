Module 1: Similarity Calculator

Step 1
Cosine Similarity = Σ(R_u,i * R_v,i) / (||R_u|| * ||R_v||)

Calculates similarity coefficients between Alice and all other users. Highlighted users are the selected top neighbors (3).

Module 2: Candidate Generator

Step 2
Candidate Generation Rule: Extracts unique items that the chosen top similar users rated/liked, but the target user Alice has not interacted with.

Module 3: Candidate Scorer & Ranker

Step 3
Score(u, i) = Σ(Sim(u,v) * R_v,i) / Σ(|Sim(u,v)|)

Scores candidates using similarity-weighted ratings, ranks them descending, and selects the top recommendations.
